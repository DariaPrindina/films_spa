# VK Films SPA

Многостраничное SPA-приложение для просмотра фильмов на `React + TypeScript`.

## Что реализовано

- каталог фильмов с бесконечной подгрузкой по 50 карточек;
- фильтры по жанрам, рейтингу и году выпуска;
- синхронизация фильтров и поискового запроса с `searchParams`;
- детальная страница фильма;
- избранное с сохранением после перезагрузки;
- кастомное модальное окно подтверждения перед добавлением в избранное;
- сравнение до двух фильмов с автоматическим вытеснением первого при выборе третьего.

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Скопировать переменные окружения:

```bash
cp .env.default .env
```

3. Заполнить `KINOPOISK_API_KEY` в `.env`.

Для локальной разработки `Vite` проксирует запросы к Kinopoisk через `/api/kinopoisk`, поэтому ключ не уходит из браузера. Если нужен другой backend/proxy, можно задать `VITE_KINOPOISK_API_URL`.

4. Запустить приложение:

```bash
npm run dev
```

5. Собрать production-версию:

```bash
npm run build
```

## Переменные окружения

- `KINOPOISK_API_KEY` - серверный ключ доступа к API для dev-proxy.
- `VITE_KINOPOISK_API_URL` - базовый URL API или вашего backend/proxy.
